# Fiction

Fiction is a Codex skill package for planning, drafting, revising, and continuity-checking the Chinese long-form xianxia web novel 《逆骨问仙》.

The current canon follows 方辰, a village youth from 云溪村. His world begins as grounded pastoral life: drought, fields, herbs, family, old channels, and village water disputes. The cultivation world is hidden at first. Fang Chen accidentally obtains the 太虚种, a quiet but powerful seed-like treasure whose early miracles appear through dew, plant vitality, roots, dreams, and village crises.

## What This Skill Helps With

- Drafting chapters for 《逆骨问仙》
- Creating chapter briefs and volume outlines
- Keeping 方辰, 林清禾, 云溪村, and 太虚种 continuity consistent
- Maintaining fast-paced serial hooks and practical爽点
- Tracking foreshadowing, hidden-world reveals, and viewpoint knowledge limits

## Usage

Install or copy `skills/write-xianxia-epic` into your Codex skills directory, then invoke:

```text
Use $write-xianxia-epic to write the next chapter of 《逆骨问仙》.
```

Example prompt:

```text
用 $write-xianxia-epic 续写第三章，节奏要更紧，每章要有现实压力、太虚种收益和新的危险钩子。
```
